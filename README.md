# ML-Research

Репозиторий для исследовательских проектов, связанных с машинным обучением, в рамках подготовки к позиции ML Researcher (сентябрь 2026, Университет Иннополис, программа "Математические основы искусственного интеллекта").

## Цели:

- Документирование процесса изучения и выполнения исследовательских задач по ML

- Воспроизведение ключевых работ (статей, экспериментов, state-of-the-art подходов)

- Развитие навыков PyTorch, исследования, публикации результатов

- Формирование портфолио из проектов (Kaggle, самостоятельные эксперименты, лабораторные задачи)

## Структура:

- `projects/` — отдельные исследовательские проекты (мини-проекты, воспроизведения статей)

- `notebooks/` — Jupyter ноутбуки с экспериментами и анализом

- `scripts/` — рабочий код и утилиты

- `docs/` — материалы, заметки, обзоры статей

- `requirements.txt` — зависимости для запуска проектов

- `README.md` — файл с объяснением цели и содержания проекта
